Vim操作

vi ../../.txt  打开文件 
保存并退出 Vim：
	按 i 进入插入模式
	按 Esc 键切换到普通模式
	输入 :wq（write + quit）
	按 Enter
放弃修改：
	按 Esc
	输入 :q!
	按 Enter





一、模式切换
	操作	命令	说明
	普通模式	Esc		进入普通模式
	插入模式	i		光标前插入
	插入模式	I		行首插入
	插入模式	a		光标后插入
	插入模式	A		行尾插入
	插入模式	o		当前行下新开一行插入
	插入模式	O		当前行上新开一行插入
	可视模式	v		进入字符可视模式
	可视模式	V		行可视模式
	可视模式	Ctrl+v	块选择模式
	命令模式	:		进入命令行模式
二、光标移动
	命令	说明
	h	左移一个字符
	l	右移一个字符
	k	上移一行
	j	下移一行
	0	移到行首
	^	移到行首第一个非空字符
	$	移到行尾
	w	移到下一个单词开头
	e	移到当前/下一个单词结尾
	b	移到当前/上一个单词开头
	gg	移到文件开头
	G	移到文件末尾
	H	移到屏幕顶部
	M	移到屏幕中间
	L	移到屏幕底部
	Ctrl+f	向前翻页
	Ctrl+b	向后翻页
	Ctrl+d	向下半页
	Ctrl+u	向上半页
	%		跳转到匹配的括号
三、文本操作
	1. 删除
		命令	说明
		x	删除光标所在字符
		X	删除光标前字符
		dd	删除整行
		D	删除光标到行尾
		dw	删除到下一个单词开头
		d$	删除到行尾
		d^	删除到行首第一个非空字符
		dG	删除到文件末尾
		dgg	删除到文件开头
		daw	删除一个单词（包括空格）
		diw	删除一个单词（不包括空格）
	2. 复制/剪切
		命令	说明
		yy	复制整行
		y$	复制到行尾
		yw	复制到下一个单词开头
		yaw	复制一个单词（包括空格）
		diw	剪切一个单词（不包括空格）
		dd	剪切整行
	3. 粘贴
		命令	说明
		p	粘贴到光标后/下行
		P	粘贴到光标前/上行
	4. 替换
		命令	说明
		r<char>			替换光标所在字符为 <char>
		R				进入替换模式（覆盖原字符）
		:s/old/new/g	替换当前行所有 old 为 new
		:%s/old/new/g	替换全文件所有 old 为 new
		:%s/old/new/gc	替换全文件所有 old，提示确认
	四、撤销与重做
		命令	说明
		u		撤销上一次操作
		U		撤销当前行所有操作
		Ctrl+r	重做撤销操作
	五、搜索与跳转
		命令	说明
		/pattern	向下搜索 pattern
		?pattern	向上搜索 pattern
		n		跳到下一个匹配
		N		跳到上一个匹配
		*		搜索光标下单词向下
		#		搜索光标下单词向上
		f<char>	在本行查找字符 <char>
		F<char>	在本行查找字符 <char>（反向）
		t<char>	光标移动到字符 <char> 前一个位置
		T<char>	光标移动到字符 <char> 后一个位置
		%		跳转到匹配括号
六、窗口操作（分屏）
	命令	说明
	:split filename	水平分屏打开文件
	:vsplit filename	垂直分屏打开文件
	Ctrl+w s	水平分屏
	Ctrl+w v	垂直分屏
	Ctrl+w w	切换窗口
	Ctrl+w h/j/k/l	在分屏间移动光标
	Ctrl+w q	关闭当前窗口
七、文件操作
	命令	说明
	:e filename	打开文件
	:w			保存文件
	:w filename	另存为
	:wa			保存所有文件
	:q			退出
	:q!			强制退出（不保存）
	:wq			保存并退出
	:x			保存并退出（同 :wq）
	:set nu		显示行号
	:set nonu	隐藏行号
八、宏操作
	命令	说明
	q<char>	录制宏到寄存器 <char>
	q		停止录制
	@<char>	执行寄存器 <char> 的宏
	@@		重复上一次宏
九、折叠操作
	命令	说明
	za	折叠或展开当前折叠
	zc	折叠当前折叠
	zo	展开当前折叠
	zr	减少折叠等级
	zm	增加折叠等级
	zR	打开所有折叠
	zM	折叠所有折叠
十、寄存器操作
	命令	说明
	"ay	复制到寄存器 a
	"ap	从寄存器 a 粘贴
	"_d	删除不保存到寄存器（黑洞寄存器）
十一、辅助命令
	命令	说明
	:!command	执行外部 shell 命令
	:r filename	读取文件内容到当前光标
	:set paste	打开粘贴模式（避免缩进混乱）
	:syntax on	开启语法高亮
	:syntax off	关闭语法高亮
	:help keyword	查看帮助
